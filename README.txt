This repository contains the deoxybyte libraries, complete with their
dependencies, as a convenient means of providing these to other
projects.

The libraries provided are:

 - deoxybyte-gzip      https://github.com/keithj/deoxybyte-gzip.git
 - deoxybyte-io        https://github.com/keithj/deoxybyte-io.git
 - deoxybyte-run       https://github.com/keithj/deoxybyte-run.git
 - deoxybyte-systems   https://github.com/keithj/deoxybyte-systems.git
 - deoxybyte-unix      https://github.com/keithj/deoxybyte-unix.git
 - deoxybyte-utilities https://github.com/keithj/deoxybyte-utilities.git

 - cffi                http://common-lisp.net/r/projects/cffi/cffi.git
 - alexandria          https://github.com/keithj/alexandria.git
 - babel               https://github.com/keithj/babel.git
 - trivial-features    https://github.com/keithj/trivial-features.git

 - cl-fad              http://weitz.de/cl-fad/

A library directory that contains a release number in its name
e.g. cl-fad-0.6.4 contains a copy of a source release. All other
library directories contain a git submodule.

Licensing is described in the individual libraries' documentation. See
each libraries' ASDF configuration file for their version numbers.
